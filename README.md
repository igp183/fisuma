## Getting Started

**Prerequisites:** Node.js 20+. Ask a maintainer for the `.env.local` file if you need to connect to the database.

```bash
git clone https://github.com/igp183/fisuma
cd fisuma
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).
